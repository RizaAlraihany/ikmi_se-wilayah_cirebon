## 2. DESIGN SYSTEM

### 2.1 Warna Resmi IKMI

| Nama         | Hex     | Penggunaan            |
| ------------ | ------- | --------------------- |
| IKMI Dark    | #001769 | Header, Footer, Judul |
| IKMI Primary | #0072C6 | Tombol, Link, CTA     |

#### Heading

Bold – Extra Bold

16px

### 2.3 CTA Hierarchy

#### Primary Button

Digunakan untuk:

- Daftar
- Bergabung
- Simpan
- Tambah Data

Style:

- Background #0072C6
- Text putih
- Radius 999px

---

#### Secondary Button

Digunakan untuk:

- Detail
- Lihat Selengkapnya
- Pelajari

Style:

- Outline
- Transparan
- Border biru

---

### 2.4 Layout Rules

#### Desktop

Max Width:

1200px

Container Center

---

#### Mobile

Komponen horizontal wajib swipe:

- Slider Pengurus
- Kartu Pilar
- Statistik Dashboard

Dilarang:

- Menumpuk banyak card ke bawah
- Membuat pengguna scroll terlalu panjang

---

### 2.5 Section Transition

Gunakan:

- Gradient Putih → Biru Muda
- Wave Divider
- Blur Gradient

Hindari garis potong kaku antar section.
